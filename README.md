# EBUDDY BACKEND REPO

## Usage
`pnpm install`

`cp .env.sample .env`

Important! Customize the following env according to your settings

CORS_ALLOW_ORIGIN=http://localhost:3000 <= your frontend local url

FIRESTORE_PROJECT=ebuddy-73033

FIRESTORE_EMULATOR_HOST=127.0.0.1:8080

`pnpm dev`

OR

`pnpm build`

`pnpm start`
